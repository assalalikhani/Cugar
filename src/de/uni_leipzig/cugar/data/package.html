Provides the classes which implement methods for input/output and methods to get access to all adapted algorithms e.g.{@link de.uni_leipzig.bf.cluster.BorderFlow}.

{@link de.uni_leipzig.cvk.data.EdgeListGraphReader} and {@link de.uni_leipzig.cvk.data.GraphMLReaderMod} are used by the {@link de.uni_leipzig.cvk.data.Model} to read in files. Both classes use the {@link de.uni_leipzig.cvk.data.GraphReaderSettings} interface. The {@link de.uni_leipzig.cvk.data.Model} has a {@link de.uni_leipzig.cvk.data.cluster.ClusterContext} instance to get access to all adapted algorithms in package {@link de.uni_leipzig.cvk.data.cluster}.

Only the {@link de.uni_leipzig.cvk.data.Model} class is used by other classes outside of this package.
